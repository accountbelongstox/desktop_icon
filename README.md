# desktop_icon

#### 介绍
一个桌面管理软件,开发环境node.jsv18.10.0

#### 软件架构
Node.js - electron v21.1
通信方式 contentBridge/IpcMain/IpcRrenderer

#### 安装教程


#### 使用说明
将桌面图标文件放于 D:/ProgramData/Desktopicon_Backup,并使用文件夹进行分组
启动运行软件即可
